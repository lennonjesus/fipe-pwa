# FIPE-PWA [Site](https://lennonjesus.github.io/fipe-pwa)

Progressive Web Application (PWA) for Tabela FIPE Motos

### Get

```
git clone https://github.com/lennonjesus/fipe-pwa.git
```

### Install

```
npm install
```

### Run

```
npm start
```

### Build for production
```
npm run build
```
